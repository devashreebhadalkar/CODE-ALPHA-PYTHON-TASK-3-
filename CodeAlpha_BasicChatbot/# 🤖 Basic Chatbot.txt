# 🤖 Basic Chatbot

## Description

A simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

The chatbot responds to predefined user inputs such as greetings and simple questions.

## Features

* Greeting responses
* Simple conversation flow
* Exit command using "bye"
* Console-based chatbot

## Concepts Used

* Functions
* Loops
* If-Elif Statements
* Input/Output

## How to Run

1. Open terminal.
2. Run:

python chatbot.py
